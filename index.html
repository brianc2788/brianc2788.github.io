How to use:
Due to the smaller size of my projects, repositories have been organized by programming language and contain directories for other projects and their resources within. Due to the contrast with traditional Git repositories, I am issuing this disclaimer. Project directories may contain plaintext source code and/or binaries built for the x86/x64/ARM ISAs. Most are ELF with the exception of many C/Cpp projects, of which contain PE formatted executables compiled with MSVC or MinGW. A few contain binaries for ARM; either 32-bit (armv7l) or 64-bit (armv8+). Some with portability & sharing in mind, some not. All are licensed under the Apache v2 OSS license and are distributed as-is.
